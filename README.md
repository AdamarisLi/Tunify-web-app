# Tunify-web-app
Desarrollo de una aplicación web que permite a los usuarios explorar artistas musicales, descubrir nuevos géneros y acceder a rankings oficiales de Spotify.  La plataforma ofrece funcionalidades como búsqueda de artistas por nombre, navegación por géneros musicales y visualización de información detallada de cada artista.

## Funcionalidades
- Búsqueda de artistas
- Navegación por géneros
- Visualización de perfiles
- Integración con Spotify

## Tecnologías
- HTML
- CSS
- Java
- SQL
